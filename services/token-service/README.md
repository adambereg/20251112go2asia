# Token Service

Сервис учёта Points (off-chain токенов).

## Функции

- Балансы Points пользователей
- История транзакций
- Начисление и списание Points

## Разработка

```bash
# Локальная разработка
pnpm dev

# Миграции БД
pnpm db:migrate:generate
pnpm db:migrate:up

# Деплой
pnpm deploy
```
