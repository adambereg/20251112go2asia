# Content Service

Сервис контента для Atlas, Pulse и Blog модулей.

## Функции

- Управление странами, городами, местами
- События и мероприятия
- Статьи и новости

## Разработка

```bash
# Локальная разработка
pnpm dev

# Миграции БД
pnpm db:migrate:generate
pnpm db:migrate:up

# Деплой
pnpm deploy
```
