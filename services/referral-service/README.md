# Referral Service

Сервис реферальной программы.

## Функции

- Генерация реферальных кодов
- Учёт реферальных связей
- Статистика рефералов

## Разработка

```bash
# Локальная разработка
pnpm dev

# Миграции БД
pnpm db:migrate:generate
pnpm db:migrate:up

# Деплой
pnpm deploy
```
